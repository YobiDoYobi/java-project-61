## Setup

```bash
gradle installDist
```

## Run

```bash
./build/install/app/bin/app
```

## Description
1. Choose game number. 
2. Enter your name. 
3. You have 3 rounds. You need enter 3 right answers to win.
4. To escape enter "exit"
